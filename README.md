# 小红书世界杯免比分

一个 Chrome 扩展，用来隐藏小红书世界杯页面里的比分、完场状态、进球/战报类标题、积分榜、进球时间线和技术统计，同时保留对阵双方与回放/集锦入口。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`
2. 打开右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择这个目录：

   `/Users/ziho/Desktop/vibe/xhs-worldcup-spoiler-shield`

5. 重新打开或刷新小红书世界杯页面

## 覆盖范围

- 世界杯主页：`https://www.xiaohongshu.com/worldcup26`
- 单场回放页：`https://www.xiaohongshu.com/worldcup26/match/...`

扩展弹窗里可以单独开关「比分/状态」「集锦标题」「剧透面板」。
